## üö® Troubleshooting & Support

### DevBot isn't responding in Slack
**Common solutions:**

1. **Check permissions:** Ensure @DevBot has access to the channel
2. **Reconnect:** Run `/devbot reconnect` in Slack
3. **Check quota:** Verify you haven't exceeded token limits
4. **Contact support:** support@tolani.dev

### Code generation quality issues
**Improve results with better prompts:**

- Be specific: "Build REST API for user management" vs "build API"
- Include context: Mention your tech stack and requirements
- Provide examples: Reference existing code patterns
- Iterate: Ask DevBot to refine the generated code

### Integration problems
**Most integration issues resolve with:**

1. **Re-authorize:** Reconnect the integration in settings
2. **Check permissions:** Ensure proper API access
3. **Update tokens:** Refresh expired API keys
4. **Documentation:** Check integration guides at docs.tolani.dev
